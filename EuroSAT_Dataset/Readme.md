## DATASET DESCRIPTION

The EuroSAT dataset is a large collection of satellite images gathered from the Sentinel-2 satellite.
It is widely used for land use and land cover classification tasks.

It contains images from different classes that represent types of terrain or land cover.

Dataset Details:

- Total Images: ~27,000
- Image Size: 64 × 64 pixels
- Spatial Resolution: 10m – 60m (varies by band)
- Number of Bands: 13 spectral bands

  ---
  ## Class Distribution
  
| Class Name        |         Description       |
| ----------------- | ------------------------- |
| **AnnualCrop**    | Fields of seasonal crops  |
| **Forest**        | Dense tree cover          |
| **HerbaceousVeg** | Low-lying vegetation      |
| **Highway**       | Roads and highways        |
| **Industrial**    | Built up industrial areas |
| **Pasture**       | Grassland for grazing     |
| **PermanentCrop** | Orchards and plantations  |
| **Residential**   | Houses, buildings         |
| **River**         | River or waterway         |
| **SeaLake**       | Seas or lakes             |
